创建仓库

仓库名字为 github 用户名.github.io

生成SSH key

  1. ssh-keygen -t rsa -C “GitHub注册邮箱”

  2. ssh git@github.com

    如果出现The authenticity of host xx can’t be established

    输入yes

  3. 进入~/.ssh 中复制 id_rsa.pub文件内容

  4. 头像 - setting - SSH and GPG keys

  5. Title可以随意填，将复制的内容添加进Key
