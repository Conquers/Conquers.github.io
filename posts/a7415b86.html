安装

  • 在线编写，请百度搜索 Overleaf

  • 在本地编写 LaTex 文件，需要安装 texlive（TeX发行版） + texstudio（文本编辑器或其他编写工具）

texlive

texlive 下载地址：Index of /CTAN/systems/texlive/Images/ | 清华大学开源软件镜像站 | Tsinghua Open Source Mirror

windows 安装方式：安装包下载完后，打开下载后的安装包，双击运行 install-tl-windows.bat 耐心等待安装，不要点 abort，一直等到出现安装完成。

texstudio

textstudio 下载地址：TeXstudio - A LaTeX editor (sourceforge.net)

以上都安装完成后，即可在 texstudio 中编写 LaTex 格式的文件。

可选项：texlive + vscode + git

  1. 在 vscode 中下载 LaTex Workshop 插件

  2. 创建文档、编译及查看

  1. 在同级目录创建 git 仓库

Mac

Mac 下载 Tex 发行版地址：MacTeX - TeX Users Group (tug.org)

其他的可以参考 Windows 安装。

问题

vscode 编译报错 error: spawn latexmk ENOENT

如果第一次使用 vscode ，请检查环境变量！

如果之前正常用，突然不可以了，请重启电脑！

之前用 vscode 都可以用，突然用不了，找了很久的答案，还是在 stackoverflow 上找到的，只需要重启就可以了。

visual studio code - Recipe terminated with fatal error: spawn latexmk ENOENT - Stack Overflow

导入了 subfigure，但是编译报错

报错信息如下

检查导言区是否有这两个包，如果是，注释调第一个，即可解决

citation undefined

点击 texstudio 中的工具，选择清理辅助文件，再次点击工具，选择参考文献，再重新编译

如果不能解决，试试下面的方法，不同的电脑可能解决的方式不一样

  • TexStudio提示报错citation undefined的解决_texstudio citation undefined-CSDN博客
  • LaTex编译时出现undefined citation的解决办法。_latex citation undefined-CSDN博客

编译报错，在某个 figure 块内报错

具体报错信息如下：I can't write on file 'xxxx.pdf'. \includegraphics{xxx}

这是因为在其他软件预览了编译后的 pdf，关闭即可
